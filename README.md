# DevOps SPA Lab - React

This is a simple Single Page Application (SPA) project for the DevOps Lab.

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Build the project:
   ```sh
   npm run build
   ```
3. Start the development server:
   ```sh
   npm start
   ```

## Deployment
This project can be deployed using GitHub Pages.
